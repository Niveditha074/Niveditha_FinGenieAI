# FinGenieAI

Full Stack Financial Management Application built using ReactJS and Spring Boot.

## Features
- User & Admin Login
- JWT Authentication
- Accounts Management
- Transactions Management
- Loan Management
- Investments
- Fraud Alerts
- EMI Calculator
- Credit Score Analysis
- Risk Checker
- QR Payment Simulation
- AI Chat Assistant

## Tech Stack
Frontend: ReactJS, Axios, React Router
Backend: Spring Boot, Spring Security, JWT, JPA, Hibernate
Database: MySQL

## Project Structure
- frontend/
- backend/
- screenshots/

## Run Backend
mvn spring-boot:run

## Run Frontend
npm install
npm start
